# 证言 Testimony.ai Web Demo

## 🛡️ AI深伪时代青少年霸凌事件可信存证系统

### 项目概述

**证言（Testimony.ai）** 是一款面向青少年校园霸凌与网络暴力受害方的AI可信存证系统。在AI深度伪造技术使"眼见不再为实"的时代，通过端侧多源时间锚定、可审计AI引导式事件回顾、多模态交叉验证三项核心技术，帮助受害者在黄金时间窗口内独立完成一份可被法庭采纳的原始证据固定。

### 核心功能

| 模块 | 功能描述 | 技术实现 |
|------|---------|---------|
| **伪装计算器** | App伪装为计算器图标 | 真实可用计算器 + 秘密手势触发 |
| **安全空间** | 一键进入录屏+时间锚定+通知静音 | 四源交叉校验(系统/NTP/基站/区块头) |
| **AI引导访谈** | 五阶段司法级引导流程 | FSM有限状态机 + 15条诱导性检测规则 |
| **家长端分析** | 观察录入 + 风险评估 + 沟通脚本 | AI隐私隔离架构 |
| **证据包生成** | 司法级加密证据包 | Merkle树 + SHA-256哈希链 |

### 快速启动

```bash
# 方式1：直接打开
直接用浏览器打开 index.html 即可运行

# 方式2：本地HTTP服务
cd web-demo
python3 -m http.server 9000
# 访问 http://localhost:9000
```

### 使用指南

#### 1. 进入系统
1. 打开页面后看到**伪装计算器界面**
2. **连续点击左上角区域6次**（或快速点击屏幕任意位置6次）
3. 输入PIN码：**123456**
4. 选择身份：**学生** 或 **家长**

#### 2. 学生端流程
```
选择学生模式 → 启动安全空间 → AI引导访谈 → 查看证据包 → 司法报告
```

#### 3. 家长端流程
```
选择家长模式 → 输入观察内容 → AI智能分析 → 获取风险评估和沟通脚本
```

### API配置

系统已内置API密钥，支持实时AI对话。如需修改：

1. **右键点击Logo/盾牌图标** → 打开API配置面板
2. 或编辑 `app.js` 中的 `AppState.apiConfig`

```javascript
apiConfig: {
    apiKey: 'your-api-key',
    endpoint: 'https://api.qnaigc.com/v1',
    model: 'deepseek-v3'
}
```

### 技术亮点

#### 🔒 安全特性
- **AES-256-GCM加密** - 硬件级密钥存储(Android Keystore)
- **四源时间锚定** - 系统/NTP/基站/区块链交叉验证
- **Merkle树完整性校验** - 增量更新避免ANR
- **诱导性检测15条规则** - AST级别实现

#### 🤖 AI能力
- **有限状态机(FSM)** - 5个核心状态，全程可审计
- **创伤知情对话协议** - 稳定化→信息收集→赋能三阶段
- **降级机制** - AI输出必须经规则引擎过滤
- **连续3次拦截触发专家接管**

#### 👨‍👩‍👧 隐私保护
- **数据严格隔离** - 家长端绝对无法查看学生原始记录
- **未成年人完全自主** - 学生拥有查看、删除、分享权
- **端侧优先** - 数据默认本地加密存储
